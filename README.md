<h2>How to init?</h2>

```bash
# To clone the repository
$ git clone https://github.com/FrancescoGM/structure-of-a-project-node

# To install all dependencies
$ yarn install
# or
$ npm i

# To run the project
$ yarn dev
# or
$ npm dev 
```
